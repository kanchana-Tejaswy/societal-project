# Plastic Waste Management System (Sub-Project)

> [!IMPORTANT]
> This is the core application directory. For the full, comprehensive 10,000-word project documentation including AI architecture, database schemas, and environmental impact studies, please refer to the root [README.md](../README.md).

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run the server: `python app.py`
3. View the app: `http://127.0.0.1:5000`

## Key Files
- `app.py`: Main Flask controller.
- `ml_model.py`: TensorFlow CNN interface.
- `database.py`: SQLite persistence layer.
- `static/`: Modern UI assets.
- `templates/`: Jinja2 HTML templates.
